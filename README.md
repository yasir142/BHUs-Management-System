BHUs Management System (Final Year Project)



A desktop-based Healthcare Management System developed using C# and Windows Forms.





🏥 Project Background



This project is developed as a university Final Year Project and is based on a simulated Basic Health Unit (BHU) system. It is designed for academic purposes to demonstrate how a primary healthcare management system can digitize patient records, staff management, and medicine inventory in a controlled environment.





Features



\- Patient Registration

\- Staff Management

\- Medicine Records

\- Login System





🛠️ Technologies Used



\- C#

\- Windows Forms (.NET Framework)

\- SQL Server

\- Visual Studio
